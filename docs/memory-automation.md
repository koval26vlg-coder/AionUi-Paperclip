# Автоматизация общей памяти

## Что автоматизировано

Настроен фоновый наблюдатель:

```text
tools/watch-memory.ps1
```

Он отслеживает изменения в:

- `AGENTS.md`
- `README.md`
- `.cursor/`
- `.kiro/`
- `docs/`
- `tools/`

Когда меняются правила, задачи, решения, журналы или слои памяти, наблюдатель автоматически пересобирает:

```text
docs/context-packs/context-pack-latest.md
```

## Автозапуск

Автозапуск оформляется через Windows Task Scheduler.

Имя задачи:

```text
Aion File Memory Auto
```

Текущий статус: установлен и запущен. Проверка выполнялась 2026-05-10.

Задача должна запускаться через PowerShell 7:

```text
C:\Program Files\PowerShell\7\pwsh.exe
```

Причина: часть скриптов и документов содержит русский UTF-8. Старый Windows PowerShell 5.1 может читать UTF-8 без BOM как ANSI и ломать русские строки в `.ps1`.

Установить автозапуск:

```powershell
.\tools\install-memory-autostart.ps1
```

Проверить статус:

```powershell
.\tools\status-memory-auto.ps1
```

Удалить автозапуск:

```powershell
.\tools\uninstall-memory-autostart.ps1
```

## Логи

Лог автоматизации:

```text
logs/memory-auto.log
```

## Что остается обязанностью агента

Автоматизация не может сама понять смысл работы, если агент ничего не записал.

Агент все равно должен:

1. Работать по `AGENTS.md`.
2. Оставлять отчет в `docs/agent-log/` или через MCP-инструмент `add_agent_log`.
3. Сохранять важные долгосрочные факты в `docs/memory/layers/` или через MCP-инструмент `add_memory`.
4. Создавать handoff, если работу должен продолжить другой агент.

После этого система сама обновит общий контекстный пакет.

## Важное ограничение

Автоматизация не может автоматически вытащить закрытую историю чата из Codex, Cursor, Kiro или другого приложения, если агент не записал результат в общую базу.

Поэтому обязательное правило остается прежним: после работы агент должен сохранить отчет, память или handoff.

## Принцип

Пользователь не должен вручную нажимать кнопку для обновления памяти.

Достаточно, чтобы агент записал результат в общую базу. Наблюдатель сам пересоберет контекст.
