# Общая память

Эта папка описывает будущую общую память для Codex, Cursor и Kiro.

## Рекомендуемый вариант

Graphiti MCP / Zep Knowledge Graph:

- локальная память;
- граф связей между задачами, решениями, файлами и агентами;
- подходит для долгого проекта;
- лучше сохраняет историю изменений и контекст.

## Альтернатива

Mem0 / OpenMemory MCP:

- проще подключить;
- удобные инструменты добавления и поиска воспоминаний;
- может требовать внешний аккаунт или API.

## Временный режим до подключения MCP

Пока MCP-память не подключена, общей памятью считаются:

- `docs/current-context.md`
- `docs/decisions.md`
- `docs/tasks.md`
- `docs/agent-log/`

Каждый агент обязан читать эти файлы перед работой и обновлять после работы.
