# AionPaperclipWorkspace

Эта папка является общей рабочей областью для трех агентов:

- Codex
- Cursor
- Kiro

Цель рабочей области: сохранить общий контекст, решения, задачи и результаты работы так, чтобы каждый агент мог понять, что делали другие, и дать оценку их работе.

## Как пользоваться

1. Открывай эту папку в Codex, Cursor и Kiro.
2. Перед задачей проси агента прочитать `AGENTS.md` и `docs/current-context.md`.
3. После задачи агент должен добавить запись в `docs/agent-log/`.
4. Важные решения записываются в `docs/decisions.md`.
5. Текущая картина проекта поддерживается в `docs/current-context.md`.

Подробная инструкция с готовыми фразами находится в `docs/how-to-use-agents.md`.

## Быстрый запуск

Запусти файл:

```text
OPEN-AGENT-WORKSPACE.cmd
```

Он попытается открыть эту папку в Cursor и Kiro, а также запустить Codex CLI в отдельном PowerShell-окне внутри этой рабочей папки.

## Роли агентов

Codex используется как основной инженер: сложный анализ, правки, тесты, ревью и аккуратное изменение файлов.

Cursor используется как IDE-агент: быстрые правки, локальная навигация по проекту, ручная разработка и второе мнение.

Kiro используется как архитектор и планировщик: specs, требования, дизайн, задачи, steering и контроль качества перед реализацией.

## Общий принцип

Ни один агент не является единственным источником истины. Источником истины являются файлы в `docs/` и правила в `AGENTS.md`.
