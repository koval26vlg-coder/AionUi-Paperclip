# Задача

Проверить hierarchical workflow на реальной задаче поиска авиабилетов.

Исходный запрос пользователя: найти самые дешевые авиабилеты туда-обратно для двух взрослых с вылетом из аэропорта Волгограда (VOG) в аэропорт Манилы (MNL), с июля 2026 по конец 2026 года, длительность поездки примерно неделя, допускается +/- пару дней. Рабочая трактовка: вылет 2026-07-01..2026-12-31, длительность 5..9 ночей, 2 взрослых, валюта RUB.

Критерий успеха: дать пользователю самый дешевый найденный вариант и несколько резервных вариантов, явно отделить подтвержденные факты от ограничений источника, не искажать постановку на переходах между уровнями.

Источники/артефакты: календарный артефакт Aviasales `D:\AionUi-Paperclip\docs\agent-workflows\flight-vog-mnl-aviasales-calendar-2026-07-12-two-adults.json`.

Ограничение: точная ticket-search выдача Aviasales в текущем окружении возвращала 403, поэтому календарные цены нужно считать предварительными. Calendar API не раскрывает явный passenger-count; оценка для двух взрослых считается как `calendar_price * 2` и должна быть проверена на странице поиска/checkout перед покупкой.
