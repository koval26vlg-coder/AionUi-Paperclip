## Какие есть риски

- Calendar API не раскрывает явный passenger-count; `estimated_two_adults_rub = price_calendar_rub * 2` является предварительной оценкой.
- Exact ticket-search endpoint Aviasales в текущем окружении вернул 403, поэтому точная выдача не подтверждена автоматически.
- Многие даты не дали calendar rows; нельзя утверждать абсолютный минимум по всему рынку, только минимум среди доступного calendar surface.
- Цена динамическая и может измениться до покупки.
