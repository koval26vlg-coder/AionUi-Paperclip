## Какие есть риски

1. **Риск неполноты данных**: Из-за отсутствия данных по 93% дат в календаре нельзя гарантировать, что июль — это абсолютно единственный период с дешевыми билетами во втором полугодии. В реальном поиске на другие даты могут быть более выгодные предложения.
2. **Риск тарифов для группы**: Оценка 147 828 RUB рассчитана как `цена * 2`. В реальности при наличии только одного места по минимальному тарифу общая стоимость для двоих пассажиров на checkout будет выше.
3. **Риск изменения цен и 403 ошибки**: Попытка прямого обращения к API точной выдачи Aviasales возвращает HTTP 403 Forbidden. Проверку придется делать вручную или через веб-интерфейс (веб-версию Aviasales).
4. **Риск транзита и багажа**: Дешевые билеты на дальние направления часто включают смену аэропортов, несколько пересадок (например, в Китае PVG) или не включают багаж, что критично для недельной поездки.
