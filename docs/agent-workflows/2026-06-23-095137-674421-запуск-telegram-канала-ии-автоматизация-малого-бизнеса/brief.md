# Запуск Telegram-канала: ИИ + Telegram-автоматизация для малого бизнеса

Пользователь хочет "залететь" в нишу Telegram-канала и начать делать проект, используя локальный hierarchical workflow.

## Исходный выбор ниши
Рабочая ниша: ИИ и Telegram-автоматизация для малого бизнеса, самозанятых, экспертов, владельцев каналов и SMM-специалистов.

## Цель workflow
Подготовить практически исполнимый запуск Telegram-канала: позиционирование, название/обещание, рубрики, контент на первые дни, план роста, первая монетизация и критерии проверки спроса.

## Важные ограничения
- Не делать финансовые/инвестиционные обещания и не уходить в токсичный инфобизнес.
- Не запускать длительные collectors/backtests/background jobs.
- Не писать во внешние системы без отдельного подтверждения пользователя.
- Не использовать секреты/API-ключи.
- Все выводы и handoff вести на русском.

## Ожидаемый результат
- Четкая концепция канала.
- 7-дневный контент-план.
- 3-5 вариантов названия.
- Первый лид-магнит/мини-продукт.
- План набора первых подписчиков.
- Метрики, по которым через 7-14 дней решить: продолжать, сузить или менять нишу.
