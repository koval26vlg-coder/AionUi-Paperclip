# Что нужно подтвердить для внешнего запуска

Я подготовил локальные файлы. Чтобы перейти к внешним действиям, нужно явное подтверждение пользователя.

## Вариант A. Пользователь запускает вручную

Подтверждение не нужно. Используйте файлы из `launch-bundle`:

- `01-channel-profile.txt`;
- `02-pinned-post.txt`;
- `03-lead-magnet-google-doc.md`;
- `04-day-1-posts.txt`;
- `05-leads-tracker.csv`.

## Вариант B. Агент помогает создать канал/документ

Нужно отдельное сообщение пользователя, например:

```text
Подтверждаю внешние действия: помоги создать Telegram-канал "ИИ в дело" и Google Doc для лид-магнита. Публикуй только после моего подтверждения каждого текста.
```

## Что будет внешним действием

- создание Telegram-канала;
- публикация закрепа;
- публикация постов;
- создание Google Doc/Notion;
- изменение доступа по ссылке;
- покупка рекламы или размещений.

Без такого подтверждения эти действия не выполняются.
