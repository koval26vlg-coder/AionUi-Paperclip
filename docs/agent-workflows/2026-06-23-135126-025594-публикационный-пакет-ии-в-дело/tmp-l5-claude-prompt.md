# Voice profile
# Voice profile канала "ИИ в дело"

## Роль автора

Автор говорит как практик, который каждый день собирает простые автоматизации для людей без технической команды.

Не как инфобизнес-тренер.
Не как новостник про нейросети.
Не как разработчик, который усложняет.

## Тон

- Спокойный.
- Прямой.
- Конкретный.
- На "вы".
- Без давления.
- Без обещаний "заработаете x10".
- Без слов "революция", "взрыв", "уникальная методика", "секретный способ".

## Формула поста

1. Одна боль.
2. Почему она возникает.
3. Простое решение.
4. Шаблон, сценарий или список действий.
5. Мягкий CTA.

## Лексика

Использовать:

- "рутина";
- "заявка";
- "клиент";
- "шаблон";
- "сценарий";
- "первый шаг";
- "минимальная версия";
- "не усложняйте";
- "проверьте руками";
- "если повторяется, можно автоматизировать".

Не использовать:

- "воронка на миллион";
- "легкие деньги";
- "прорыв";
- "секрет";
- "гарантированный рост";
- "пассивный доход";
- "нейросети сделают все за вас".

## Длина

Обычный пост: 900-1600 знаков.

Структура:

- короткий первый абзац;
- блоки по 1-3 строки;
- списки там, где это ускоряет чтение;
- один CTA в конце, не в каждом посте.

## CTA

Мягкие CTA:

- "Напишите 'шаблон', если хотите пример."
- "Напишите 'карта', если хотите разобрать вашу Telegram-воронку."
- "Ответьте одним словом: что у вас сейчас больше всего съедает время?"
- "Сохраните, если собираете бота или форму заявок."

## Главный редакционный фильтр

Перед публикацией спросить:

```text
После этого поста человек может что-то сделать сегодня?
```

Если нет - пост не готов.

# Setup kit excerpt
# Channel setup kit для "ИИ в дело"

## Название

Основной вариант:

```text
ИИ в дело
```

Запасные варианты:

- ИИ без рутины
- Бот вместо рутины
- Автоматизация для дела
- Telegram без хаоса

## Описание канала

Короткая версия:

```text
Практичные ИИ-сценарии, Telegram-боты и шаблоны для малого бизнеса, экспертов и самозанятых.
Одна боль - одно решение - один шаблон.
```

Расширенная версия:

```text
ИИ в дело - канал про простую автоматизацию без технической команды.

Пишу про Telegram-ботов, промпты, заявки, контент, follow-up и мини-воронки для малого бизнеса, экспертов и самозанятых.

Без хайпа. Только то, что можно применить сегодня.
```

## Username идеи

Проверить вручную в Telegram:

- `ii_v_delo`
- `ai_v_delo`
- `iivdelo`
- `bot_bez_rutiny`
- `ai_bez_rutiny`
- `telegram_bez_haosa`

## Закрепленный пост

```text
ИИ в дело

Здесь я показываю, как малому бизнесу, экспертам и самозанятым убирать рутину с помощью ИИ, Telegram-ботов и простых шаблонов.

Формат канала:

1. Одна боль.
2. Простое решение.
3. Промпт, сценарий или шаблон.
4. Понятный следующий шаг.

Без новостной ленты про нейросети.
Без обещаний легких денег.
Без сложной разработки ради разработки.

Стартовый материал:
10 Telegram-автоматизаций без программиста.

Чтобы получить, напишите "автоматизация".
```

## Первые действия при создании канала

1. Создать канал с названием `ИИ в дело`.
2. Поставить короткое описание.
3. Добавить закрепленный пост.
4. Скопировать лид-магнит в Google Doc, Notion или PDF.
5. Подготовить таблицу учета заявок.
6. Опубликовать первые 3 поста до приглашения людей.
7. Проверить, что CTA ведет к понятному действию.

## Таблица учета заявок

Колонки:

```text
Дата
Имя
Username
Откуда пришел
Что попросил
Боль
Следующий шаг
Статус
```

Статусы:

- попросил лид-магнит;
- попросил шаблон;
- задал вопрос;
- интерес к карте;
- консультация;
- неактуально;
- вернуться позже.

## Как перенести лид-магнит в Google Doc

1. Создать новый документ.
2. Название: `10 Telegram-автоматизаций без программиста`.
3. Скопировать текст из `lead-magnet-doc-ready.md`.
4. Сделать заголовки `Heading 1/Heading 2`.
5. Добавить в начало короткое описание.
6. В конце оставить CTA: "напишите карта".
7. Включить доступ по ссылке только на просмотр.
8. Проверить ссылку в режиме инкогнито.

## Как перенести в Notion

1. Создать страницу.
2. Вставить markdown-текст.
3. Проверить списки и кодовые блоки.
4. Добавить блок "Как пользоваться".
5. Включить публичный доступ.
6. Проверить ссылку.

## Как сделать PDF

Минимальный путь:

1. Скопировать в Google Doc.
2. Отформатировать заголовки.
3. Добавить простую обложку.
4. Export -> PDF.
5. Проверить на телефоне.
