# Контент-пакет для Telegram-канала "ИИ в дело"

Пользователь попросил выполнить первый практический шаг после выбора ниши: написать первые 15-20 постов и собрать лид-магнит "10 Telegram-автоматизаций без программиста", используя workflow.

## Исходный контекст
Предыдущий workflow `2026-06-23-095137-674421-запуск-telegram-канала-ии-автоматизация-малого-бизнеса` завершен. Итоговая ниша: прикладной канал "ИИ в дело" — ИИ и Telegram-автоматизация для малого бизнеса, самозанятых, экспертов, владельцев каналов и SMM.

## Цель workflow
Создать готовый стартовый контент-пакет:
- 20 Telegram-постов для первых 2-3 недель;
- лид-магнит "10 Telegram-автоматизаций без программиста";
- порядок публикации и простые CTA;
- без внешних публикаций, рекламы, создания канала, форм или ботов.

## Ограничения
- Не использовать токсичный инфобизнес и обещания легкого заработка.
- Не публиковать во внешние системы без отдельного подтверждения.
- Не использовать секреты/API-ключи.
- Не запускать долгие collectors/background jobs.
- Учитывать, что MiMo/Antigravity runtime для workflow handoff ранее был нестабилен; fallback должен быть записан явно.

## Критерии готовности
- Есть файл с 20 готовыми постами.
- Есть файл лид-магнита, который можно перенести в Google Doc/Notion/PDF.
- Есть финальный отчет workflow.
- Проверен статус workflow `done`.
