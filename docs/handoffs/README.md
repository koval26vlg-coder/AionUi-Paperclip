# Передачи между агентами

Эта папка нужна для случаев, когда один агент завершил работу, а другой должен продолжить.

Пример:

- Codex сделал реализацию.
- Cursor должен проверить.
- Kiro должен оформить spec.

В таком случае Codex создает handoff-файл в этой папке.

Используй шаблон `docs/templates/handoff.md`.
