# Журнал работы агентов

Здесь хранятся отчеты Codex, Cursor и Kiro.

Один файл - одна рабочая сессия или один значимый результат.

Рекомендуемое имя файла:

```text
YYYY-MM-DD-HHMM-agent-task.md
```

Примеры:

```text
2026-05-10-1430-codex-setup-context.md
2026-05-10-1510-cursor-review-codex.md
2026-05-10-1600-kiro-auth-spec.md
```

Используй шаблон `docs/templates/agent-report.md`.
