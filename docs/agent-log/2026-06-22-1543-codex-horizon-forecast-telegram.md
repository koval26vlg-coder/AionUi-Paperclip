## Решения

- Основной много-дневный сценарий теперь `/forecast`, а не ручной просмотр `/best` по каждому матчу.
- Горизонт ограничен 1-5 днями, потому что пользователь просил 2-5 дней и дальше качество локального снимка/линии быстро устаревает.
- Реальные ставки/клики в БК не автоматизировались.

## Риски и ограничения

- Прогноз использует текущий локальный снимок линии и xG. Перед реальным решением нужно обновлять модель видимым `/model`.
- У матчей без xG выбор может опираться только на 1X2 из signals, без дополнительных голевых рынков.
- Самый вероятный исход часто имеет низкий коэффициент; поэтому отдельно показывается баланс вероятность/кэф.
