## Запрос
Добавить реальный машинный источник xG для прироста над Elo.

## Результат
FBref заблокирован Cloudflare (не машинно-доступен). Найден рабочий источник: xgscore.io рендерит прогнозный xG по матчам ЧМ в DOM. Извлечены реальные xG 9 матчей -> data/manual/match_xg.csv. Создан src/xg_model.py: Пуассон от xG -> вероятности 1X2, blend_probabilities, load/lookup. Интегрировано в core.evaluate_signals (параметр match_xg, settings xg_weight=0.35) и cli (--match-xg). Модель теперь = Elo(published) + xG(Пуассон) + рынок. Демонстрация сдвига: Португалия-Узбекистан Elo/рынок ~80%, с xG(2.1-1.1) -> 71% (xG видит, что Узбекистан создаёт больше моментов). Колумбия-ДРКонго 65%->59%. 30 тестов OK. Запуск: tools/run-wc-model.ps1 (xG авто-подключается). Кандидатов 0 — рынок эффективен, xG уточняет, а не выдумывает edge.
