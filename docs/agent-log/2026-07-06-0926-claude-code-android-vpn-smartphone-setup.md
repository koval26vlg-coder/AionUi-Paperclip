## Риски и ограничения

- Link/QR — секреты, не публиковать и не пересылать через мессенджеры/облака.
- Один клиентский UUID используется и, возможно, будет использоваться на нескольких
  устройствах — для Xray VLESS это допустимо, но при желании можно завести отдельный
  UUID для телефона через `tools\vpn\deploy_android_xray_reality.ps1`.
