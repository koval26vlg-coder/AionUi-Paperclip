## Исходный запрос пользователя

Пользователь спросил, какую букмекерскую контору выбрать для ставок, и попросил рассмотреть вариант Polymarket.

## Контекст перед началом

- Активный betting analytics контур остается manual decision-support: агент дает прогноз и порог коэффициента, пользователь принимает решение вручную.
- Active run gate `trading_mvp` оставался `RUNNING`; долгие процессы не запускались.

## Что сделано

- Проверены актуальные источники: ФНС, рейтинги/списки легальных БК РФ, официальные страницы Polymarket/Polymarket US, CFTC.
- Подтверждено: Polymarket ограничивает доступ из России и запрещает VPN-обход; для РФ не рекомендовать как рабочий вариант.
- Практическая рекомендация: не выбирать одну БК навсегда, а держать 2-3 легальные БК для line shopping; базово рассматривать Fonbet/Winline/PARI/Marathon/Betcity из легальных операторов, проверяя лицензию в ФНС и платежи через ЦУПИС.

## Риски и ограничения

- Это не юридическая, финансовая или азартно-игровая рекомендация.
- Условия БК, лимиты и доступность меняются; перед регистрацией проверять официальный реестр ФНС и ЦУПИС.
- Не использовать Polymarket из геозапрещенной юрисдикции и не обходить ограничения.
